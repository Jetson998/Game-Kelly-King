# 竞拍小游戏项目

暂定名：**捡漏之王**

这是一个“竞拍 + 鉴定 + 转卖 + NPC 心理博弈”的轻量小游戏项目。

## 目录说明

- `docs/01-game-design.md`：游戏策划案
- `docs/02-development-plan.md`：分阶段开发步骤
- `docs/03-context-strategy.md`：龙虾/OpenClaw 对话上下文控制方案
- `docs/04-mvp-spec.md`：第一版 MVP 规格
- `docs/05-data-design.md`：数据结构与数值设计
- `docs/06-task-backlog.md`：可执行任务清单
- `src/`：后续开发文件目录
- `assets/`：后续美术、音效、字体等资源目录

## 当前目标

先不急着写完整游戏代码，先把项目拆小：

1. 确定 MVP 范围
2. 搭建最小项目结构
3. 实现一场可玩的文本/网页拍卖 Demo
4. 再逐步加入 UI、NPC、鉴定、出售、成长系统

## 推荐开发原则

- 每次只开发一个小模块
- 每次提交前能运行/能验证
- 不在一次对话里塞太多需求
- 先玩法，后美术
- 先单机，后扩展排行榜/存档/每日挑战
