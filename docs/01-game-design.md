# 游戏策划案：《捡漏之王》

## 1. 一句话概念

玩家从小额资金起步，参加不同等级的拍卖会，低价拍下未知价值的物品，再通过鉴定和转卖赚取利润，最终成为拍卖圈大佬。

核心爽点：

- 低价捡漏
- 高价转卖
- 判断 NPC 是真想买还是在抬价
- 在有限资金下做取舍
- 偶尔暴赚，偶尔买贵破防

## 2. 核心循环

### 单场循环

1. 进入拍卖场
2. 展示拍品
3. 查看有限线索
4. 与 NPC 竞价
5. 成交或放弃
6. 鉴定真实价值
7. 选择出售渠道
8. 获得盈利或亏损

### 长线循环

跳蚤市场 → 二手商会 → 古董拍卖 → 神秘拍卖 → 终极拍卖赛

## 3. MVP 版本范围

第一版只做一个能验证乐趣的小 Demo：

- 1 个拍卖场：跳蚤市场
- 20 个拍品数据
- 3 个 NPC
- 7 天挑战模式
- 基础竞价
- 基础鉴定
- 基础出售
- 现金、库存、结算

暂不做：

- 复杂剧情
- 技能树
- 多地图
- 联网排行
- 大量美术
- 复杂黑市系统

## 4. 玩家初始状态

```text
现金：1000
库存上限：3
目标：7 天结束时现金达到 10000
失败：现金为 0 且库存为空
```

## 5. 拍品类型

第一版拍品类型：

- 旧手机
- 老相机
- 纪念币
- 二手游戏机
- 旧手表
- 手办
- 球鞋
- 老唱片
- 小古董
- 神秘盒子

## 6. NPC 类型

### 新手小白

- 估值误差大
- 容易乱加价
- 偶尔误打误撞抢到好东西

### 二手贩子

- 对普通商品比较准
- 出价理性
- 不太追高

### 抬价托

- 未必真想买
- 会把价格推高
- 到一定价位可能突然放弃

## 7. 关键差异化

重点做 **NPC 心理提示**：

```text
二手贩子突然坐直了，也许这东西有价值。
抬价托笑得太明显，可能是在坑你。
新手小白连续加价，看起来有点上头。
```

玩家不是单纯算数值，而是在判断人。

## 8. 第一版胜负

- 7 天后现金 ≥ 10000：胜利
- 现金为 0 且没有库存：失败
- 其他情况：普通结算，显示财富等级
